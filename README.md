# FE25-JS2-slutprojekt-back-jack-gustafsson
Backend-delen för slutprojektet

Nedan finns en lista med node-kommandon som kan användas vid användning av repot:
<ol>
  <li><strong>npm i</strong> för att installera alla dependencies och devDependencies som finns listade i package.json</li>
  <li><strong>npm run dev</strong> för att testa projektet i utvecklingsläge</li>
  <li><strong>npm run build</strong> för att bygga projektet för produktion</li>
  <li><strong>npm start</strong> för att starta appen i produktion</li>
</ol>
